# 🎯 Accordion Menu

A clean and responsive Accordion Menu built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**.

This project demonstrates how to create an interactive accordion component using JavaScript DOM manipulation with a fully responsive layout for desktop, tablet, and mobile devices.
